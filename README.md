# Tiny Renderer Zig RayLib 

The project was initially setup using the raylib language bindings for zig found [here](https://github.com/raylib-zig/raylib-zig). The language bidning repository can be used to create a own project by executing `project_setup.sh project_name`.

In order to run the application from source use `zig build run`.

This is a work in progress software renderer using Zig and RayLib.

It is based on the tinyrenderer collection of articles found here. The tinyrenderer articles form a course with the goal of implementing a software renderer in native c++.
